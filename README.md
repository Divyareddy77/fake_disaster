# 🌀 Fake Disaster Tweet Detection

This app uses a Machine Learning model to classify tweets as **real** or **fake** disaster alerts.

## ⚙️ Technologies Used

- Python
- Streamlit
- scikit-learn
- TfidfVectorizer + Logistic Regression
